Getting started

Dependencies

Ng-boosted is an extension of ng-bootstrap. It uses Orange graphic chart. It allows you to run your Angular project based on Boosted CSS. Like ng-bootstrap that backports js component from bootstrap in angular, ng-boosted aimed to backport custom Orange component of boosted in Angular.

Here are the dependencies that we rely on and you don't need to explicitly add to your project:

  • Boosted {{globals.boostedVersion}}
  • Ng-bootstrap {{globals.ngBootstrapVersion}}

Beware the versions above-mentioned are the minimal required versions. Some components need ng-bootstrap import.

Installation

For ng-bootstrap installation, please read its documentation.

Install ng-boosted via either npm or yarn:

npm install --save ng-boosted
yarn add ng-boosted

Once installed, you need to import ng-boosted modules.

import {{ '{' }} NgBoostedModule {{ '}' }} from 'ng-boosted';

@NgModule({{ '{' }}
 declarations: [AppComponent, ...],
 imports: [ NgBoostedModule, ...],
 bootstrap: [AppComponent]
{{ '}' }})
export class AppModule {{ '{' }}
{{ '}' }}

If you need to use a component from ng-bootstrap, you have to import it like explained in the ng-bootstrap documentation.

From version 5.1.3

Delete NgBoostedModule and add:

import {{ '{' }} OCalendarModule, OCarouselModule, ODropdownModule, ONavLinkModule, ONavMenuModule, ONavbarModule, ScrollTopModule, OSwitchModule, OToastModule {{ '}' }} from 'ng-boosted';

@NgModule({{ '{' }}
 declarations: [AppComponent, ...],
 imports: [ OCalendarModule, OCarouselModule, ODropdownModule, ONavLinkModule, ONavMenuModule, ONavbarModule, ScrollTopModule, OSwitchModule, OToastModule, ...],
 bootstrap: [AppComponent]
{{ '}' }})
export class AppModule {{ '{' }}
{{ '}' }}

Adding Boosted CSS

Three options can be used to import Boosted CSS when installed from npm.

Configure your angular.json

Add the following path:

"styles": [
    "node_modules/boosted/dist/css/boosted.min.css",
    "node_modules/boosted/dist/css/orangeIcons.css"
]

Add orangeIcons.css, if you need.

Import Boosted styles

Import Boosted minified CSS:

@import '../node_modules/boosted/dist/css/boosted.min.css'

Or import Boosted SCSS sources:

@import '../node_modules/boosted/scss/boosted.scss'

Building this documentation

Simply npm run build:docs and the output goes to a _gh_pages folder.

You can also run this documentation with npm run start.

The output is then served to http://localhost:4200.
