Dependencies

Ng-boosted is an extension of ng-bootstrap. It uses Orange graphic chart. It allows you to run your Angular project based on Boosted CSS. Like ng-bootstrap that backports js component from bootstrap in angular, ng-boosted aimed to backport custom Orange component of boosted in Angular.

Here are the dependencies that we rely on and you don't need to explicitly add to your project:

  • Boosted
  • Ng-bootstrap

Installation

Install ng-boosted via either npm or yarn:

npm install --save ng-boosted
yarn add ng-boosted

Once installed, you need to import ng-bootstrap & ng-boosted modules.

Ng-bootstrap module

    import {{ '{' }} NgbModule {{ '}' }} from '@ng-bootstrap/ng-bootstrap';
    @NgModule({{ '{' }}
     ...
     imports: [ NgbModule, ...],
     ...
    {{ '}' }})
    export class AppModule {{ '{' }} {{ '}' }}

For more information, please read its documentation.

Ng-boosted module

Under version 5.1.3 :


    import {{ '{' }} NgBoostedModule {{ '}' }} from 'ng-boosted';
    @NgModule({{ '{' }}
     ...
     imports: [ NgBoostedModule, ...],
     ...
    {{ '}' }})
    export class AppModule {{ '{' }} {{ '}' }}

From version 5.1.3 to version 5.2.0 :

NgBoostedModule deleted.

You just need to import the component module you want to use.


    import {{ '{' }} ScrollTopModule, ... {{ '}' }} from 'ng-boosted';
    @NgModule({{ '{' }}
     ...
     imports: [ ScrollTopModule, ...],
     ...
    {{ '}' }})
    export class AppModule {{ '{' }} {{ '}' }}

Warning !

From version 5.2.0 :

You don't need to import ng-boosted modules. The ng-bootstrap module will be sufficient.

You can no longer use the below components, they have been removed.

  • Calendar : lib-o-calendar
  • Navbar : lib-o-navbar
  • o-nav-menu & o-nav-link
  • Scroll Top : llib-scroll-top
  • Swiper : lib-o-carousel
  • Switch : lib-o-switch
  • Toast : lib-o-toast
  • Toggle

Here are the equivalent components :

  • Datepicker
  • navbar
  • Dropdown
  • Scroll Up
  • Carousel
  • -
  • Toast
  • Buttons

Adding Boosted CSS

Configure your angular.json

Add the following path:

"styles": [
    "node_modules/boosted/scss/boosted.scss",
    "node_modules/boosted/dist/css/orangeIcons.css",
    "node_modules/boosted/dist/css/orangeHelvetica.css",
    "node_modules/ng-boosted/style/ng-boosted.scss"
    ]

Add orangeIcons.css & orangeHelvetica.css, if you need.

Warning !

Add ng-boosted.scss From version 5.2.0.

Building this documentation

Simply npm run build:docs and the output goes to a _gh_pages folder.

You can also run this documentation with npm run start.

The output is then served to http://localhost:4200.
