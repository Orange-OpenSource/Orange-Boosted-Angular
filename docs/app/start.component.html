Getting started

Dependencies

Ng-boosted is an extension of ng-bootstrap. It uses Orange graphic chart. It allows you to run your Angular project based on Boosted CSS. Like ng-bootstrap that backports js component from bootstrap in angular, ng-boosted aimed to backport custom Orange component of boosted in Angular.

Here are the dependencies that you don't need to explicitly add to your project:

  • Boosted {{globals.boostedVersion}}
  • ng-bootstrap {{globals.ngBootstrapVersion}}

Beware the versions above-mentioned are the minimal required versions. Some components need ng-bootstrap import. Find below instructions.

Installation

For ng-bootstrap installation, please see the documentation here.

Install ng-boosted via:

npm install --save ng-boosted

or

yarn add ng-boosted

Once installed, you need to import ng-bootstrap, if you use it, and ng-boosted modules.

import {{ '{' }} NgbModule {{ '}' }} from '@ng-bootstrap/ng-bootstrap';
import {{ '{' }} NgBoostedModule {{ '}' }} from 'ng-boosted';

@NgModule({{ '{' }}
 declarations: [AppComponent, ...],
 imports: [NgbModule.forRoot(), NgBoostedModule, ...],
 bootstrap: [AppComponent]
{{ '}' }})
export class AppModule {{ '{' }}
{{ '}' }}

If you need to use a component from ng-bootstrap you need to import it like explained in the ng-bootstrap documentation here. import {{ '{' }} NgbModule, NgbActiveModal {{ '}' }} from '@ng-bootstrap/ng-bootstrap';

Adding Boosted CSS

Three options can be done to import Boosted CSS if downloaded from npm

Configure your angular.json

Inside the file, add the following path:
"styles": [
"/path/node_modules/boosted/dist/css/boosted.min.css",
"styles.css"
]

Configure your styles.css

Inside your styles.css, import boosted css:
@import '~boosted/dist/css/boosted.min.css'

Configure your styles.scss with SASS

if you are using SASS, then you need to do this in your styles.scss:
@import '../node_modules/boosted/scss/boosted.scss'

Building this documentation

Simply npm run build:docs and the output goes to a _gh_pages folder.

You can also run this documentation with npm start

The output is then served to http://localhost:3030
